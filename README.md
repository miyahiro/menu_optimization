# 献立の最適化

本リポジトリは、Pulpを用いた献立の最適化用のリポジトリです。

## ファイル、フォルダ構成

1. [README.md](./README.md)

   本ファイル

2. [problem_definition.md](./problem_definition.md)

   献立の最適化の問題設定及び定式化について記述したファイル

3. [problem.py](./problem.py)

   問題インスタンス用クラスを定義しているモジュール

4. [solution.py](./solution.py)

   最適化結果を格納するクラスを定義しているモジュール

5. [solver.py](./solver.py)

   問題インスタンスを受け取り、それに対する最適化結果を計算するクラスを定義しているモジュール

6. [sample.py](./sample.py)

   サンプルの問題インスタンスを作成し、それに対する最適化結果を表示する実行可能モジュール

7. [requirements.txt](./requirements.txt)

   必要となるPythonモジュールを記述したファイル
